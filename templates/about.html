About Software Builder

Software Builder is a local LLM coding agent with a built-in evaluation system, designed for software engineers who want full control and transparency.

Technology Stack

Backend

  • Flask - Lightweight web framework
  • SQLAlchemy - ORM for database management
  • Alembic - Database migrations
  • SQLite - Local database storage
  • Pydantic - Type-safe configuration
  • structlog - Structured logging

Frontend

  • Datastar - Hypermedia framework for reactive UI
  • Pico.css - Minimal CSS framework
  • Server-Sent Events - Real-time updates

Development Tools

  • uv - Fast Python package manager
  • Ruff - Lightning-fast linter and formatter
  • pytest - Testing framework
  • just - Command runner

Key Features

🤖 Coding Agent

Specialized prompts and tools for software engineering tasks. The agent can read files, write code, execute tools, and maintain context across conversations.

📊 Built-in Evaluation

Comprehensive evaluation framework based on industry best practices:

  • Trace Collection - Automatic capture of all interactions
  • Manual Annotation - Binary pass/fail judgments
  • Dataset Management - Build test cases from failures
  • Prompt Versioning - Track performance across versions
  • Automated Tests - pytest integration for CI/CD

🔒 Local & Private

Everything runs on your machine. Your code, conversations, and data never leave your system unless you explicitly use a cloud LLM API.
